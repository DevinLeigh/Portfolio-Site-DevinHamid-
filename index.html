My Work

Mix Tape

The task of this project was to create a UI direction for a music player app that would work for branding and digital UI of the product with a target audience age range between 35-55 years old. Falling within that age range myself, I set out to create a useful, fun product that I would personally enjoy using.

Spot Finder

Tasked with creating a responsive web app of my choosing for location-based recommendations and completing both the UX and UI phases of the design, I went from competitive analysis and user research all the way through to the style guide and final screen mockups. The result was an app that would help dog owners find locations for themselves where their pets could join them.
